Estufa Inteligente

Automatize seu cultivo com sensores Arduino

Tecnologia que cultiva por você

A estufa automatizada com sensores Arduino ajusta a abertura do teto conforme o clima, mantendo suas plantas sempre protegidas e saudáveis.

Como Funciona

Sensor de Luminosidade

Detecta a intensidade do sol e abre o teto automaticamente.

Controle Sustentável

Automação energética eficiente com Arduino.

Sensor de Temperatura

Monitora o calor ambiente e aciona alertas para evitar extremos.

Sensor de Umidade

Garante o nível ideal de umidade para o crescimento saudável das plantas.

Display LCD

Exibe em tempo real as informações dos sensores para fácil visualização.

Alerta Inteligente

Receba notificações automáticas no celular sobre qualquer anomalia detectada.

Especificações Técnicas

  • Estrutura em alumínio galvanizado
  • Sensor DHT22 (temperatura e umidade)
  • Microcontrolador Arduino Uno
  • Servo motor para abertura do teto
